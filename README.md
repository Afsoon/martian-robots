# Martian Robots

In the file `index.test.js` you can find two suites:
- The suite `Turn and move commands`: This is compose of unit testing of move and turn commands.
  - If the robot is allowed to move.
  - If the robots is correctly facing after execute a move command.
  - If the robot has been lost or not and his last position.
- The suite `Process robost commands and input`: This is compose of integration and unit test.
  - The test named `We receive the full input and we process all robots` is the integration
We have a 100% coverage of the code.

This project was made with Vainilla Javascript and Jest.

## How to use the project

- Execute `npm install`
- After `npm run test`
